Bring back "Open command window here" context menu item in Windows

2018-06-27 1 min read

It’s a simple registry hack.

  1. Strat registry editor: Run -> regedit.
  2. Go to HKEY_CLASSES_ROOT\Directory\shell\cmd.
  3. Right-click the cmd (folder) key, and click Permissions.
  4. Click the Advanced button.
  5. Advanced Security Settings dialog will open, at the top next to Owner click Change.
  6. Put in your account name, click Check Names to make sure domain etc is set up correctly, click OK.
  7. Check the Replace owner on subcontainers and objects check box near Owner field.
  8. Click Apply, OK.
  9. In the now open Premissions dialog, select Administrators, check Full Control checkbox below.
  10. Click Apply, OK.
  11. You are now back in the registry editor. Click the cmd folder on the left.
  12. Look for HideBasedOnVelocityId key in the right window tab.
    • If it’s there, Right-click, Rename to ShowBasedOnVelocityId.
    • If it’s not there, just add the ShowBasedOnVelocityId key (type DWORD), value 0x00639bc8 (6527944).
  13. Now try Shift + Right Click in an Explorer window, you should see the Open command window here option.
